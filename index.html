        🏆 Format & trades
        ▾
      
      

        
Standings format

        

          
          
        

        

          
Managers play a round-robin each round; the winner (higher round score) takes
            the log points, plus bonuses — an attacking bonus for a big score and a losing bonus for a narrow defeat.

          

            
            
            
          

          

            
            
          

          
          
          

            
How a rumble round scores:

            

              
              
            

            
Head-to-head vs all: you play every rival that round, banking the usual win/draw/loss points.
              Placement: managers are ranked by round score and the top finishers earn the points below (ties split evenly).

            
          

          

        

        
Trades & free agents

        

          
          
        

        
Instant: free-agent swaps apply immediately (first-come). Waiver: claims queue and
          resolve when the window closes, in reverse-standings priority (last place picks first).

        
        
Applies to both modes — the number of free agents a manager can bring in
          each window. Instant: swaps beyond the cap are blocked. Waiver: only that many claims are awarded per window.

        
Captain

        

          
          
        

        
Each manager names a captain (double points that round) and a vice-captain
          (doubles instead if the captain doesn't play at all). Set them in the lineup screen.

        
Substitutions

        
        
Trade & lineup windows

        

          
          
        

        
Automatic mode reads the competition's matchweek calendar: the trade window
          opens 1h after a matchweek's last game and closes 24h before the next one; lineups lock 1h before the next
          matchweek's first kick-off. Best with a fixture-driven competition like the Premier League.

          Competition & players
          
        
          Pick a competition to draft from. Its teams, squads and
          fixtures are pulled from API-Football once and shared by every league on the same
          competition — if another league already loaded it, this reuses that data instantly (no
          re-pull), and one admin's stat pull updates them all. Leave unset for the built-in World
          Cup 2026 squads. Season = the API's season key (a domestic league's start year, e.g. 2024
          for 2024/25; a tournament's year).
          
        
        
          
        
          
          
          
        
          
        
        
          
      
          
      
          
        
          Scheduled pulls · app admin
          
        
          Turn the automatic daily stat pull on or off per competition.
          When on, the scheduled job pulls that competition's completed games into the shared stats each
          day — one pull serves every league on it. When off, use "Pull stats now" by hand.
          
        
          
      
          
      
          
        
          Pull stats now
          
        
          Fetches completed matches for the chosen date straight
          from API-Football and writes everyone's stats — same as the 06:00 automated pull, on
          demand. In-progress games are skipped; re-pull after the final whistle.
          
        
          
          
          
        
          
        
          
      
          
      
          Manual stats entry
          
      
          Fallback alongside the automated daily pull. Rows upsert on
        (player, match), so re-saving edits in place. Use the same match label format as the
        automation: Home vs Away (YYYY-MM-DD).

          Trading window & lineup locks
          
        
          Open it between rounds (once all teams have played);
          managers can then trade, swap free agents and change their lineups. Closing it
          locks everyone's lineup in for the next games — points for each matchday are
          scored against the lineup that was locked at the time, so later changes never
          rewrite earlier rounds. Close it before kickoff.
          
        
          
          
          
        
          
        
      
          
      
          
        
          Draft settings

          Switch this off for a blind draft where nobody can rank the pool by numbers.
          
        
          
          
          
        
          
      
          
      
          
        
          ⚙️ Design your draft — scoring & bonuses
          
        
          Customise how points are scored for this league. Editable until
          the first game is scored, then locked so past results never shift. Leave as-is for the standard
          World Cup values.
          
        
          
      
          
      
          
        
          Redrafts & final phase
          
        
          As the field narrows: (1) optionally remove trailing
          managers — their points freeze and their players return to the pool; (2) set the keeper
          rules and open keeper picks; (3) set the new squad size and hit Start — the redraft
          begins immediately in reverse-standings snake order, so do it when everyone's
          around, on a day with no matches. Kept players fill squad-quota slots and each
          keeper costs that manager's earliest draft round. TEAM picks always carry through.
